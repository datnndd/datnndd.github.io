# Certificates

Put certificate PDFs in this folder.

Current naming:

- `topik-3-korean.pdf` for Korean certificate.
- `english-certificate.pdf` for English certificate when available.

Each available PDF appears on the portfolio and opens through `certificate.html?file=<file-name>`, so one shared viewer page can display the whole certificate PDF from a direct link.

When the site is published from `datnndd/datnndd.github.io`, every PDF in this folder is listed automatically in the Certificates section. For richer labels, update `certificateMetadata` in `assets/js/certificates.js` and `certificates` in `assets/js/certificate-viewer.js`.
